Команды для запуска проекта

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
В проекте использовались технологии:
1. Next.js + Typescript
2. AntDesign для notifications
3. Axios
4. UseLocalStorage
5. Взаимодействие с Rest api https://fakestoreapi.com
